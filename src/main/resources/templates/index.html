WEB Addressbook

Springboot Practice Web System

アプリの特徴

  • オンラインでアドレス帳を管理できる便利な WEB アプリです。
  • お客様のお悩みにすぐに対応できるよう、専門のスタッフによるコールセンターを用意しています。
  • 何となくいい感じのデザインになるように画面の構成を考える。
  • SpringSecurity を使用してログイン機能を作成する。
  • Bootstrap4 を使用して画面を作成する。

# WEBアドレス帳（Springboot Practice）

## 概要

* アドレス情報を登録、検索、更新、削除するだけの単純なアプリにする。

<br />

## その他

* Bootstrap4 を使用して画面を作成する。

* SpringSecurity を使用してログイン機能を作成する。

* よくある感じのデザインにする。
